# She Weaves 🧵

**She Weaves** is a full-stack e-commerce platform designed to showcase and sell handmade crochet products.
The platform highlights creativity, craftsmanship, and the idea of threads connecting people through handmade gifts.

## ✨ Features

* Browse handmade crochet products
* User authentication (register & login)
* Product catalog with images and descriptions
* Add items to cart
* Place orders
* Secure payment integration (Razorpay)
* Responsive and modern UI
* Interactive thread-inspired design elements

## 🏗️ Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion (animations)

### Backend

* Java
* Spring Boot
* REST APIs
* Maven

### Database

* MySQL

### Payment Integration

* Razorpay

## 📂 Project Structure

```
sheweaves
│
├── frontend   # React + TypeScript frontend application
│
└── backend    # Spring Boot backend APIs
```

## ⚙️ How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/yourusername/she-weaves.git
```

### 2. Run Backend (Spring Boot)

* Open the **backend** folder in Spring Tool Suite / IntelliJ
* Run the Spring Boot application

### 3. Run Frontend

Navigate to the frontend folder:

```
cd frontend
npm install
npm run dev
```

The frontend will run on:

```
http://localhost:5173
```

## 🎨 Concept

The design of **She Weaves** is inspired by the idea of threads weaving together stories, creativity, and connection.
Scrolling interactions and visual elements reflect the feeling of fabric being woven across the page.
